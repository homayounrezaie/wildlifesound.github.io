# Test Sounds

Put sample audio files here so users can test WildlifeSound with the upload button.

Supported formats:

- `.mp3`
- `.wav`
- `.m4a`
- `.webm`
- `.ogg`
- `.flac`

Use short clips when possible. The app is designed for recordings up to 60 seconds.

Suggested examples:

- Bird call near trees
- Frog call near water
- Insect sounds at night
- Mixed outdoor soundscape

Only add audio files that you own or that are licensed for redistribution.
